# Cats vs Dogs Classifier

A CNN-based image classifier built using TensorFlow/Keras to classify cats and dogs.

## Features
- Built with TensorFlow and Keras
- Trained on Kaggle Dogs vs Cats dataset
- Achieved ~82% validation accuracy
- Includes training and validation graphs

## Technologies Used
- Python
- TensorFlow
- Keras
- Matplotlib
- Google Colab

## Dataset
https://www.kaggle.com/datasets/salader/dogsvscats
